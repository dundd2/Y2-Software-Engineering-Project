UI Module
=========

This module provides the classes for building the game's User Interface (UI). It includes reusable components like buttons (`UIButton`) and input fields (`UIInput`), base classes for different pages (`BasePage`), and specific page implementations for the Main Menu, Settings, Game Setup, How to Play, End Game screen, etc.
