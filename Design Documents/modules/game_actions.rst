Game Actions Module
===================

This module encapsulates specific actions that can occur during the game, often triggered by player input or game events. It acts as an intermediary between the event handler/main loop and the core game logic, providing a cleaner interface for performing actions like playing a turn, handling buy decisions, starting auctions, etc.
