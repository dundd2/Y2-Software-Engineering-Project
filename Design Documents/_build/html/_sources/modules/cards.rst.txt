Cards Module
============

This module contains classes and functions related to the game cards, including the Card and CardDeck classes that manage the Pot Luck and Opportunity Knocks cards in the game.

Key Features
------------

* **Card Types**: Implementation of the Pot Luck and Opportunity Knocks card decks
* **Card Effects**: Various card effects such as money gain/loss, movement, and special actions
* **Card Deck Management**: Shuffling, drawing, and returning cards to the deck
* **Get Out of Jail Free Cards**: Tracking of ownership and usage
* **Card Animation**: Visual representation of card drawing and effects
* **Card Factory**: Systematic creation of cards based on source data
* **Effect Execution**: Processing card effects within the game system

Card Effect Execution
---------------------

The module processes card effects through a specialized system:

.. plantuml::

   @startuml
   
   title Card Effect Execution
   
   start
   
   :Draw Card;
   
   :Display Card Text;
   
   if (Effect Type?) then (Money)
     if (Operation?) then (Gain)
       :Add Money to Player;
     else if (Operation?) then (Lose)
       :Remove Money from Player;
     else if (Operation?) then (Exchange)
       :Transfer Money Between Players;
     endif
   else if (Effect Type?) then (Movement)
     if (Movement Type?) then (Absolute)
       :Move to Specific Square;
       if (Pass Go?) then (Yes)
         :Collect Go Money;
       endif
     else if (Movement Type?) then (Relative)
       :Move Forward/Backward;
       if (Pass Go?) then (Yes)
         :Collect Go Money;
       endif
     else if (Movement Type?) then (Special)
       if (Destination?) then (Jail)
         :Move to Jail;
         :Set Jail Status;
       else if (Destination?) then (Nearest Utility)
         :Move to Nearest Utility;
       else if (Destination?) then (Nearest Station)
         :Move to Nearest Station;
       endif
     endif
   else if (Effect Type?) then (Special)
     if (Special Type?) then (Jail Card)
       :Set Get Out of Jail Free;
       :Update Card Deck;
     else if (Special Type?) then (Repairs)
       :Calculate Repair Costs;
       :Charge Player;
     else if (Special Type?) then (Birthday)
       :Collect from Each Player;
     endif
   endif
   
   :Update Game State;
   
   stop
   
   @enduml
