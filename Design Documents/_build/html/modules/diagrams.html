Game State Machine

The game uses a state machine to manage different phases of gameplay:

MENU –> PLAYING : Game Start

state PLAYING {

[*] –> WAITING_FOR_ROLL

WAITING_FOR_ROLL –> MOVING : Roll Dice MOVING –> PROPERTY_DECISION : Land on Unowned Property MOVING –> PAY_RENT : Land on Owned Property MOVING –> DRAW_CARD : Land on Card Space MOVING –> SPECIAL_SQUARE : Land on Special Square

PROPERTY_DECISION –> WAITING_FOR_NEXT_PLAYER : Buy/Pass PAY_RENT –> WAITING_FOR_NEXT_PLAYER DRAW_CARD –> CARD_EFFECT CARD_EFFECT –> WAITING_FOR_NEXT_PLAYER SPECIAL_SQUARE –> SPECIAL_EFFECT SPECIAL_EFFECT –> WAITING_FOR_NEXT_PLAYER

WAITING_FOR_NEXT_PLAYER –> WAITING_FOR_ROLL : Next Player

PLAYING –> DEVELOPMENT_MODE : Development Button DEVELOPMENT_MODE –> PLAYING : Exit Development

PLAYING –> AUCTION_MODE : Property Auctioned AUCTION_MODE –> PLAYING : Auction Complete

PLAYING –> TRADE_MODE : Trade Button TRADE_MODE –> PLAYING : Trade Complete

PLAYING –> GAME_OVER : End Condition Met

GAME_OVER –> MENU : New Game

@enduml
